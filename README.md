• Identify risky customers
• Monitor NPA
• Track Recovery

- Customer
- Branch
- Product
- Loan Amount
- EMI
- Status

## KPIs

- Total Loan Amount
- Total Customers
- NPA
- Collection Rate
- Default Rate
