# 👨‍💼 HR Analytics Dashboard

## 📌 Project Overview

The **HR Analytics Dashboard** is an interactive Business Intelligence solution developed using **Power BI** to help Human Resource teams analyze workforce performance, employee demographics, attrition trends, recruitment, and compensation. The dashboard transforms raw HR data into meaningful insights, enabling organizations to make informed workforce planning and talent management decisions.

---

# 🎯 Business Problem

HR departments often manage large volumes of employee data across multiple departments and locations. Traditional reporting methods make it difficult to identify workforce trends, monitor employee turnover, evaluate hiring performance, and support strategic HR decisions.

This dashboard centralizes HR data into a single interactive platform, allowing management to monitor workforce metrics, improve employee retention, and optimize organizational performance.

---

# 🎯 Project Objectives

- Analyze employee demographics and workforce distribution.
- Monitor employee attrition and retention trends.
- Track hiring and exit patterns.
- Evaluate department-wise employee performance.
- Analyze salary and compensation distribution.
- Monitor attendance and leave utilization.
- Support HR decision-making through interactive visualizations.
