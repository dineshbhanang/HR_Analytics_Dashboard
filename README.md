
# Loan Portfolio Dashboard

## Business Problem

Banks need to monitor loan performance, delinquency, collections, and NPAs.

## Objectives

• Analyze loan portfolio
• Identify risky customers
• Monitor NPA
• Track Recovery

## Dataset

- Loan ID
- Customer
- Branch
- Product
- Loan Amount
- EMI
- Status

## KPIs

- Total Loan Amount
- Total Customers
- NPA
- Collection Rate
- Default Rate
